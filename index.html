JDao.js

Peque√±a libreria Javascript que incorpora algunos elementos del bootstrap de twitter de una manera mas facil de usuar junto a la libreria Jquery

Version de la libreria : V 2.0

Elementos Incorporados:

  • Alertas Personalizadas
  • Ajax Simplificado

Como utilizarla

para poder utilizar la libreria JDao tienes que incluirla a tus archivos html,php,aspx,etc junto con la libreria jquery para que pueda funcionar

Ejemplo:

Archivo.html
  1. <!DOCTYPE html>
  2. <html lang="es">
  3. <head>
  4. <link href="css/bootstrap.css" rel="stylesheet"/>
  5. <script src="js/jquery.js"></script>
  6. <script src="js/JDao.js"></script>
  7. </head>
  8. <body>
  9. aqui todo el contenido que desees dentro de la pagina
  10. </body>
  11. </html>

Alertas

JDao trae un conjunto de alerta personalizadas para que las pueas utilizar en tus proyectos

ALERTA DE ERROR

alerta de tipo error para mostar cuando alla ocurrido un error
el primer parametro que recibe esta funcion es para poder colocarle el titulo a la alerta en est caso es una imagen aunque podria ser texto
segundo parametro sirve para mostrar un texto informativo en la alerta
el ultimo parametro es un numero entero unico de cualquier alerta

  1. <!DOCTYPE html>
  2. <html lang="es">
  3. <head>
  4. <link href="css/bootstrap.css" rel="stylesheet"/>
  5. <script src="js/jquery.js"></script>
  6. <script src="js/JDao.js"></script>
  7. <script>

        2
     );
  10. </script>
  11. </head>
  12. <body>
  13. <input type="botton" value="Alerta tipo error" id="btnerror">
  14. </body>
  15. </html>
